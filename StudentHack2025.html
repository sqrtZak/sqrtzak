StudentHack 2025 - University of Manchester

In April 2025, I partook in a Hackathon in Manchester. This hackathon was organised by the UniCs society.

I enjoyed working collaboratively with the team. We succeeded in winning the prize from Ping Proxies, using web scraping APIs to process bulk data in an efficient time frame. We utilised our mathematical knowledge and utilized the Central Limit Theorem. Then our computer science knowledge to embed this with API requests, multithreading, and proxy servers. We also developed an application that help users stay fit and healthy by using an AI API to Google's Gemini. We successfully integrated accessibility features like audio input (Native Gemini feature) and text-to-speech using Neuphonics API. The basis of this app was in python running a UI comprised of Qt.

I also enjoyed the networking of the event, meeting with Portswigger and developer lecturers from Google and the CEO of Ping Proxies

I would like to thank my team for the valiant effort and winning us a prize!
