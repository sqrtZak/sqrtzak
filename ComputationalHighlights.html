Minecraft Server

I created a self-hosted Minecraft server running in a Docker container on a 16GB Raspberry Pi 5, using Raspberry Pi OS Lite. I selected this hardware for its low idle power consumption, high uptime, and excellent performance. The server runs Geyser-MC, enabling cross-compatibility between Java and Bedrock clients. To secure the server, I implemented Cloudflare for DDoS protection and DNS management, along with TCP-Shield and PlayIt.gg tunneling. This setup provides strong security without exposing ports (19131 UDP and 25565 TCP).

Through this project, I applied and developed skills in:
Networking: UFW, iptables, tunneling, DNS configuration.
Security principles: Port exposure mitigation, DDoS protection.
Server management: Docker containerization, log monitoring, systemd services.

Self-Hosted Network Attached Storage

I built a Network Attached Storage (NAS) system on a Raspberry Pi 5, controlled via API calls secured with Cloudflare Zero Trust, including multi-factor authentication and DDoS protection. The system runs incremental backup software to efficiently manage data and ensure redundancy.
I developed a Python-based UI using Qt to control the NAS, providing an interface for client devices.

Through this project, I applied skills in:
Security & Networking: API security, Cloudflare Zero Trust, multi-factor authentication, DDoS mitigation.
System Management: Incremental backups, storage management.
Software Development (front-end): Python, Qt.

Custom Notes App

I have programmed a notes app using Gemini 3 Pro. After a few fixes to the code, I created a PyQt5 Python program that uses a bitmap canvas to help me write notes. I made this app for Linux, since there was no reliable ultra-lightweight alternative. This app has essential features: PDF import and export, undo, and screen-snipping features, and it is only 536 lines of Python. I think apps like this are essential for modern operating systems, which often have lots of dependencies, poor performance, and poor efficiency and lots of bugs and crashes.
I have uploaded this to a public repo on Github:
https://github.com/sqrtZak/NotesApp
